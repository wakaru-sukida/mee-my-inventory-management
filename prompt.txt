ต้องการสร้าง Web App ระบบ Inventory Management ที่มีฟังก์ชัน การรับเข้า, การจ่ายออก, มีหน้าสำหรับเก็บรายละเอียดข้อมูลสต๊อกสินค้าแต่ละตัว, มีระบบ location ที่สามารถ track สินค้าคงเหลือในแต่ละ location ได้แบบอัตโนมัติ, รวมทั้งยังมีรายงานสินค้าคงเหลือ และรายงานเคลื่อนไหว

อยากให้หน้า web application นี้มีความทันสมัย มี effect ต่างๆ ที่ทำให้น่าดึงดูดในการใช้งาน สมารถเปลี่ยน Theme สีของหน้า web app ได้




ตอนนี้ web ใช้งานได้แล้ว แต่อยากใช้งานจริง โดยฝากสร้าง Database โดยวิเคระห์ให้หน่อยว่า ถ้าอยากเริ่มต้นใช้งานฟรี สามารถใช้งานที่ไหนได้บ้าง รวมทั้งอยากให้แนะนำเว็บ Deploy ไปยัง Host ด้วย โดยฝากเลือก Host ที่ใช้งานฟรีเช่นกัน
